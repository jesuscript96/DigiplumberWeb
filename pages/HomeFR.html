Comment digiplumber fonctionne?

Réservez un appel avec un agent de Digiplumber pour faire un test

1. Sélectionnez une date dans le calendrier

2. Sélectionnez l'heure

3. Remplissez vos informations

4. Cliquez sur "planifier l'événement" pour réserver un rendez-vous avec Digiplumber
